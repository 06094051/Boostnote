# Boostnote

오픈소스 노트 앱

## Goal

그냥 글쓰는게 즐거워지셨으면 좋겠어요. :grinning:

- 타겟 OS : OSX, Windows, Linux(나중엔 모바일까지도!)
- Cloud : Google drive, Dropbox, One drive, iCloud...
- 오픈소스로 남을 것!

## 영감받은 앱/서비스

- Atom
- Quiver
- Evernote
- GitKraken
- GitBook
- Gist
- Gistbox
- Snippets Lab
